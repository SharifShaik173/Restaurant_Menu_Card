# 🍛 Biryani's Food Menu

A simple, styled HTML menu card for a Biryani restaurant — built with pure HTML & CSS.

## 📄 About

This project is a static restaurant menu webpage displaying food items across three categories:
- 🥗 **Veg**
- 🍗 **Non-Veg**
- 🍚 **Pulav**

Each section includes item names, prices (in ₹), and food images.

## 🚀 How to Run

1. Clone or download the repository
2. Open `Restaurant_menu_card.html` in any web browser
3. No installations or dependencies required!

## 🛠️ Built With

- HTML5
- CSS3 (inline + embedded styles)
